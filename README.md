
# NCAA Basketball Scoreboard

An Android app built with Kotlin and Jetpack Compose that displays NCAA Division I basketball scores by date for both men's and women's games.

## Features

- Date picker
- Men's / Women's toggle
- Game list with team names, scores, and status
- Refresh button
- Loading indicator
- Offline support with Room database caching

AI helped me with debugging code, handling offline storage, and making sure the app satisfied the assignment requirements.
